# 默认最低交易中继费率 - Default minimum transaction relay feerates

**默认最低交易中继费率**是由节点实施的政策，用于忽略那些费率低于特定金额的个别未确认交易。对于比特币核心（Bitcoin Core），这个阈值已经多年保持在 1 sat/vbyte。
